Como os três mocks se ligam

1. Tela principal Esta página — ponto de entrada para escolher loja ou admin.
2. Loja Clientes compram produtos. Admins autenticados veem botão "Painel Admin" no cabeçalho.
3. Admin Gestores acedem via botão na loja ou directamente. Link "Ver loja" no rodapé da sidebar.
